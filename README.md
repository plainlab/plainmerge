# PlainMerge

> An offline PDF mail merger

![PlainMerge](./.erb/assets/plainmerge.png)

## Features

- Drag & drop fields to customize PDF template
- Support `.xlsx`, `.xls`, `.ods` Excel files
- Support PDF forms
- Work offline
- Cross-platform
